## Install (Windows)

1. Download **wimcord-installer-0.1.4.zip** and extract it.
2. Quit Discord from the system tray.
3. Run **Install Wimcord.bat**.
4. Pick your Discord install and press Install.

Uses the official Vencord installer with Wimcord branding (UI text patched). The Wimcord build is bundled in the `wimcord` folder.

**From source:** `pnpm run build` then `pnpm run wimcord:installer`
