## Wimcord 0.1.0

First public release.

### Install (Windows)

1. Download **`Wimcord-Installer-0.1.0.exe`** below.
2. Double-click — no setup wizard.
3. Pick Discord (Stable / PTB / Canary), click **Install**, restart Discord.

Quit Discord fully before installing (system tray too).

### Included

- Full **Vencord** plugin library
- **Wimcord Panel** — settings, feature toggles, optional badges, update checks
- **Performance monitor** overlay
- **Crash diagnostics** (local logs, opt-in verbosity)
- Portable installer bundles a pre-built Wimcord `dist/`

### Notes

- Discord desktop only (not browser).
- Badge auto-register sends only your Discord user id if enabled — see [SECURITY.md](SECURITY.md).
